Aims and Objectives

There are many different aims for this application underneath is the aim and a description of the meaning behind the aims

  • Cut down waiting times via filling in an ailment form

    • This application was created to reduce waiting times of within the NHS that being at A&E or a GP surgery.This application also has a form submission system where the patient will be able to fill in an ailment form for the a doctor to process.

  • File transfer

    • This application will allow the transfer of files via wifi, 4G technology and NFC/Bluetooth from a device.

  • Combination of device data can speed up analysis

    • With the inclusion of data being retrieved from a patients device will be able to speed up the process of ailment diagnosis
